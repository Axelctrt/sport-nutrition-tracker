4. Copier le contenu du patch RC à la racine du projet sport-nutrition-tracker.
5. Accepter le remplacement des fichiers.
6. Exécuter :
   npm ci
   npm run release:verify
   npm run test:e2e
   npm run dev
7. Vérifier dans Paramètres que la version affichée est 0.16.0-rc.1.
8. Exécuter RELEASE-CHECKLIST.md sur ordinateur et iPhone 15.
9. Tester la mise à jour depuis 0.15.0, le mode hors connexion et le retour du réseau.
10. Restaurer uniquement une sauvegarde prévisualisée comme compatible.
11. Recharger avec Ctrl + F5 ou relancer la PWA si une ancienne version reste en cache.
