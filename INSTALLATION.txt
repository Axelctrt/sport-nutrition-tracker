SPORTPILOT 0.15.0-rc.1

1. Vérifier la branche release/0.15.0-rc.1.
2. Arrêter le serveur de développement avec Ctrl + C.
3. Copier le contenu du patch à la racine du projet sport-nutrition-tracker.
4. Accepter le remplacement des fichiers.
5. Exécuter :
   npm install
   npm run check
   npm run dev
6. Vérifier dans Paramètres que la version affichée est 0.15.0-rc.1.
7. Exécuter la liste complète de RELEASE-CHECKLIST.md sur iPhone.
8. Exporter une sauvegarde JSON avant tout essai de restauration ou de suppression.
9. Tester l’installation PWA, le mode hors connexion et le retour du réseau.
10. Tester un retour à l’accueil depuis l’écran global d’erreur si un incident est simulé en développement.
11. Recharger avec Ctrl + F5 ou relancer la PWA si une ancienne version reste en cache.

Schéma Dexie : version 2.
Sauvegarde JSON : version 2.
Aucune migration de données n’est réalisée.
