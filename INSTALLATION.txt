SPORTPILOT 0.15.0-alpha.10

1. Vérifier la branche feature/mobile-profile-settings-backup-ux.
2. Arrêter le serveur de développement avec Ctrl + C.
3. Copier le contenu du patch à la racine du projet sport-nutrition-tracker.
4. Accepter le remplacement des fichiers.
5. Exécuter :
   npm install
   npm run check
   npm run dev
6. Ouvrir Profil et vérifier le résumé, les trois sections et l’enregistrement sans bouton flottant.
7. Provoquer une erreur dans une section fermée et vérifier que cette section s’ouvre automatiquement.
8. Sur téléphone, appuyer sur l’icône d’engrenage de l’en-tête pour ouvrir Paramètres avancés.
9. Vérifier le résumé, le skeleton initial et les sections repliables, puis tester le changement de thème et le rétablissement des valeurs par défaut avec confirmation.
10. Ouvrir Sauvegarde et vérifier les actions Exporter et Restaurer.
11. Sélectionner une sauvegarde et vérifier le résumé avant d’ouvrir la confirmation de remplacement.
12. Ouvrir Confidentialité et stockage puis Fonctionnement de l’application.
13. Ouvrir Effacer toutes les données sans confirmer réellement, puis vérifier que le bouton reste désactivé avant la saisie EFFACER.
14. Tester en portrait et paysage, en modes clair et sombre, puis hors connexion.
15. Recharger l’application avec Ctrl + F5 si une ancienne version reste en cache.
