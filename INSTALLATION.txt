SPORTPILOT 0.15.0-alpha.1

1. Vérifier la branche feature/ux-foundations.
2. Arrêter le serveur de développement avec Ctrl + C.
3. Copier le contenu du patch à la racine du projet sport-nutrition-tracker.
4. Accepter le remplacement des fichiers.
5. Exécuter :
   npm install
   npm run check
   npm run dev
6. Tester le splash, les skeletons, les titres mobiles et le retour navigateur.
7. Recharger l’application avec Ctrl + F5 si une ancienne version reste en cache.

Le proxy Open Food Facts est actif avec npm run dev. Le build de production contacte directement Open Food Facts.
