# Tableau des accomplissements

Cette phase ajoute huit badges progressifs calculés à partir des activités, des séances de musculation terminées, des journées actives et de la variété des disciplines.

Les badges sont affichés dans les paramètres avancés avec leur progression, leur date d’obtention et le prochain objectif. Une fois gagnés, ils restent acquis dans le stockage local du navigateur, même si l’utilisateur supprime ensuite des données de test avec la réinitialisation sélective.

Aucune nouvelle table Dexie n’est ajoutée. Le schéma de base reste en version 3 et le format de sauvegarde JSON reste en version 2.
